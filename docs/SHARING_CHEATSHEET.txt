📌 ВАРИАНТ 1: ngrok (Самый популярный)

  Время: 5 минут
  Стоимость: Бесплатно
  Сложность: ⭐️ Легко

  Шаги:
  ┌──────────────────────────────────────────────────────────────┐
  │ # 1. Установка (один раз)                                   │
  │ brew install ngrok/ngrok/ngrok                               │
  │                                                              │
  │ # 2. Регистрация (один раз)                                 │
  │ https://dashboard.ngrok.com/signup                           │
  │ ngrok config add-authtoken YOUR_TOKEN                        │
  │                                                              │
  │ # 3. Запуск (каждый раз)                                    │
  │ ./share.sh                                                   │
  │ # или                                                        │
  │ ngrok http 8501                                              │
  │                                                              │
  │ # 4. Скопируй URL и отправь другу                           │
  │ https://xxxx-xxxx.ngrok-free.app                             │
  └──────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 ВАРИАНТ 2: Cloudflare Tunnel (Без ограничений)

  Время: 3 минуты
  Стоимость: Бесплатно
  Сложность: ⭐️ Легко
  Бонус: Без лимитов трафика!

  Шаги:
  ┌──────────────────────────────────────────────────────────────┐
  │ # 1. Установка (один раз)                                   │
  │ brew install cloudflare/cloudflare/cloudflared               │
  │                                                              │
  │ # 2. Запуск (каждый раз)                                    │
  │ ./share-cloudflare.sh                                        │
  │ # или                                                        │
  │ cloudflared tunnel --url http://localhost:8501               │
  │                                                              │
  │ # 3. URL появится мгновенно!                                │
  │ https://xxx.trycloudflare.com                                │
  └──────────────────────────────────────────────────────────────┘

📌 ВАРИАНТ 3: LocalTunnel (Моментальный)

  Время: 1 минута
  Стоимость: Бесплатно
  Сложность: ⭐️ Очень легко
  Минус: Менее стабилен

  Шаги:
  ┌──────────────────────────────────────────────────────────────┐
  │ # Установка и запуск                                         │
  │ npm install -g localtunnel                                   │
  │ lt --port 8501                                               │
  │                                                              │
  │ # URL:                                                       │
  │ https://random-name.loca.lt                                  │
  └──────────────────────────────────────────────────────────────┘
